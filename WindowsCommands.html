Information on Commonly used commands on Windows Command line

Command Action
cd Home Directory
cd [folder] Change directory
cd/ Root of drive
ls Short listing
chdir Displays current directory and lets you switch to other directory
path Creates and displays the path for searching executable files.
cls clear screen
rm [file] remove a file
copy copy one or more files to another location
mkdir [dir] create a new directory
rmdir [dir] remove directory
exit Ends cmd.exe
back
